== Description ==
Process your woocommerce payments using Alt-Coins, Bitcoin, Dash, ETH and other various Crypto coins including your own coin(s). Payments are handled with NodePAY.
Quick and easy setup, simply install and checkout your payment options in your woocommerce settings. NodePay merchants are paid out instantly and all blockchain functions are automated so there is no need to program or understand blockchains.

== Installation ==
Upload and activate plugin as normal. Goto Woocoomerce settings > Payments and setup your new payment gateway.

Note: You will need a free payment API key and secret to process payments - https://nodepay.uk/simple-api/
